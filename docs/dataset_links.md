# Dataset Links

## Translation
- **Training:** https://huggingface.co/datasets/Helsinki-NLP/opus-100 (config: `ne-en`)
- **Evaluation:** https://huggingface.co/datasets/facebook/flores (config: `npi_Deva-eng_Latn`)

## Summarization
- **Training + Val:** https://huggingface.co/datasets/csebuetnlp/xlsum (config: `nepali`)

## QA
- **Training + Val:** https://huggingface.co/datasets/xquad (config: `xquad.ne`)

## Why These Datasets
- opus-100: Largest clean Nepali-English parallel corpus available
- flores200: Gold standard multilingual benchmark — BLEU scores comparable to published papers
- xlsum nepali: BBC Nepali articles — highest quality Nepali text available
- xquad: Only gold-standard Nepali QA dataset
